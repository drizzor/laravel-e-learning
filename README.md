## A propos de l'application

Application fullstack réalisée avec Laravel pour le backend et InertiaJs + Vuejs pour le frontend. L'objectif était de faire une application type plateforme de formations vidéos (API de Youtube). L'autre idée était de tester le nouveau système d'authentification Jetstream venu avec la version 8 de Laravel.
